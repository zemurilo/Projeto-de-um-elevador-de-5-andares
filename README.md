# Projeto-de-um-elevador-de-5-andares
O projeto consiste no desenvolvimento de um sistema lógico para controlar um elevador de cinco níveis, usando a IDE Logisim 2.7.1 e portas lógicas de duas entradas. O circuito interpreta sensores de nível, botões internos e externos, controla o movimento do elevador e garante que ele só funcione com ambas as portas fechadas.
